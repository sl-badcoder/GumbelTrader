Fair Default-Setting Leaderboard

Gumbel Math leaderboards compare timed practice results using default settings so scores are easier to compare.

How Ranking Works

Custom settings are useful for practice, but leaderboard mode keeps the question format fixed. That means arithmetic, sequence, probability, and combinatorics scores are based on the same default setup for each mode.

Why Defaults Matter

  • Scores reflect the same timing and question configuration.
  • Practice can still be customized outside leaderboard mode.
  • Rankings reward speed and accuracy on a consistent format.
